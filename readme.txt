A simple and easy homepage slideshow for WordPress websites.

== Description ==

Upload your image(s), set the Fade time, slide time duration and the element you would like to put the slideshow in.

The images will stretch to fit the element, and will automatically resize as the window/element size changes.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `dead-simple-slideshow` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearances > Homepage Slideshow to upload the images and insert your settings.
